K · Patent box - cesion de software (art. 23 LIS)

Regla (art. 23 LIS)
Reduccion = 60% x coeficiente x renta. Coef. = gastos cedente (x1,30) / gastos totales. El numerador no puede superar el denominador. Si todo el gasto es propio, coef. = 1.
1
Calcula la reduccion del patent box
▶
euros

Coeficiente = 200.000 / 200.000 = 1,00
Reduccion = 60% x 1,00 x 260.000 = 156.000 euros
